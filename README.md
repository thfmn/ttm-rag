# Thai Traditional Medicine RAG Bot

## Quick Start Commands

### 1. Clone and setup
git clone # Todo
cd thai-traditional-medicine-rag-bot

### 2. Run setup script
chmod +x scripts/setup/install.sh
./scripts/setup/install.sh

### 3. Start services
make docker-up

### 4. Setup database
make db-setup

### 5. Install Thai language resources
make thai-setup

### 6. Run tests
make test

### 7. Start development
make dev
